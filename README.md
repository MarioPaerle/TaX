# TaX

TaX (from "ταχύ", "fast" in ancient greek), is a fast and lightweight solution
to have an ssh python IDE for when things become difficult.
It's a localhost webapp, multipage IDE and file manager I partially vibecoded since I needed it.
---
# Usage
You can simply download this github repo in a zip, extract it, and execute 
```bash
$ bash setup2.sh
```
in the folder that you need to navigate.
A venv will be created with all the requirements and the flask app will be launched.
